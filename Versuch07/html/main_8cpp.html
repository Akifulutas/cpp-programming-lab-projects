Detailed Description

Hauptanwendung zur Verwaltung von Studierendendaten.

Diese Anwendung ermöglicht das Hinzufügen, Löschen, Anzeigen, Speichern und Laden von Studentendatensätzen in einem Vektor.

Block zum Anlegen und Anhängen eines neuen Studenten am Ende.

Fügt einen neuen Studenten am Ende des Vektors hinzu. 'emplace_back' ist oft effizienter als 'push_back', da es das Objekt direkt im Vektor konstruiert und so eine zusätzliche Kopie vermeidet.

Block zum Anlegen und Einfügen eines neuen Studenten am Anfang.

Fügt einen neuen Studenten am Anfang des Vektors ein. 'studenten.begin()' ist ein Iterator, der auf die erste Position zeigt.

Sicherheitsüberprüfung: Löschen ist nur möglich, wenn der Vektor nicht leer ist.

Gibt den ersten Studenten aus. 'front()' gibt eine Referenz auf das erste Element zurück.

Löscht das erste Element im Vektor. 'erase' benötigt einen Iterator auf das zu löschende Element.

Überprüfung, ob der Vektor Elemente enthält.

Gibt den letzten Studenten aus. 'back()' gibt eine Referenz auf das letzte Element zurück.

Entfernt das letzte Element aus dem Vektor. Dies ist eine sehr effiziente Operation.

Wenn der Vektor leer ist, kann nichts gelöscht werden.

Erstellt ein temporäres Student-Objekt nur für die Suche. Nur die Matrikelnummer ist hier relevant, da der 'operator==' (in Student.cpp definiert) nur diese vergleicht.

Verwendet den 'std::find'-Algorithmus aus <algorithm>, um den Studenten zu suchen. 'std::find' nutzt intern den überladenen 'operator==' der Student-Klasse. 'it' ist ein Iterator, der auf das gefundene Element zeigt.

Überprüft, ob der 'find'-Algorithmus erfolgreich war. Wenn 'it' ungleich 'studenten.end()' ist, wurde der Student gefunden.

Löscht den Studenten an der Position, auf die der Iterator 'it' zeigt.

Gibt alle Studenten in der normalen Reihenfolge (von vorne nach hinten) aus.

Die Range-based for-Schleife ist die modernste und sicherste Methode, um über alle Elemente eines Containers zu iterieren.

Gibt alle Studenten in umgekehrter Reihenfolge aus.

Dafür werden Reverse-Iteratoren verwendet. 'rbegin()' zeigt auf das letzte Element, 'rend()' auf die Position vor dem ersten Element.
