Detailed Description

Hauptanwendung zur Verwaltung von Studierendendaten.

Dieses Programm ermöglicht das Hinzufügen, Löschen, Anzeigen und Speichern von Studierendendaten über ein textbasiertes Menü. Die Daten werden in einem std::vector<Student> gespeichert.
