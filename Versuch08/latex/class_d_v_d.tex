\doxysubsection{Detailed Description}
Die Klasse \doxylink{class_d_v_d}{DVD} ist eine konkrete Ableitung von \doxylink{class_medium}{Medium}. 

Repräsentiert eine \doxylink{class_d_v_d}{DVD} mit Titel, Altersfreigabe (FSK) und Genre. Besitzt eine eigene Ausleihlogik basierend auf der Altersfreigabe.
