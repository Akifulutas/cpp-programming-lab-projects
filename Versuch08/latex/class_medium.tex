\doxysubsection{Detailed Description}
Basisklasse für sämtliche Medien. 

Diese Klasse stellt eine Basisklasse für Objekte dar, die Medien beschreiben. Von dieser Klasse werden alle anderen Medien, wie z.\+B. \doxylink{class_buch}{Buch} oder \doxylink{class_d_v_d}{DVD} abgeleitet. Durch die rein virtuelle Funktion \textquotesingle{}ausgabe\textquotesingle{} ist diese Klasse abstrakt.
